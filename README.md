# Options Academy

A full-stack application for learning and practicing options trading with zero financial risk.

## Features
- **Interactive Tutorials:** Learn the fundamentals of options trading through structured modules and quizzes.
- **Market Explorer:** Search for real-time options chains and financial news.
- **Paper Trading:** Practice executing trades with a virtual $100,000 balance.
- **Portfolio Tracking:** Track your trade history and cash balance.

## Tech Stack
- **Frontend:** Next.js 14, Tailwind CSS, Lucide Icons, Axios.
- **Backend:** Node.js, Express, Prisma (SQLite).
- **Data APIs:** Integration ready for Alpaca, Polygon.io, and FRED.

## Getting Started

### Backend
1. `cd server`
2. `npm install`
3. `npx prisma migrate dev`
4. `npm start`

### Frontend
1. `cd client`
2. `npm install`
3. `npm run dev`
